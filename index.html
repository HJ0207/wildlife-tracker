About This Project

The Wildlife Sighting Tracker is a web application designed to manage and explore wildlife observation data from across Canada. Built with Express.js, it provides a RESTful API to query sightings by species, habitat, verification status, and more.

Available API Endpoints

Click on any endpoint below to test it in your browser!

Retrieve all wildlife sightings
GET /api/sightings Get only verified sightings
GET /api/sightings/verified List all unique species names
GET /api/sightings/species-list Find all sightings in forest habitats
GET /api/sightings/habitat/forest Search for eagle sightings
GET /api/sightings/search/eagle Get index position of moose sighting
GET /api/sightings/find-index/moose Get 3 most recent sightings
GET /api/sightings/recent
